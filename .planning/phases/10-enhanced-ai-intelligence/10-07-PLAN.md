# Plan 10-07: Mobile UI for AI Intelligence Features

## Overview
Build mobile UI components for meeting briefings, relationship suggestions, and analytics insights.

## Dependencies
- 10-06: REST & MCP endpoints

## Tasks

### 1. Create API client extensions
Update `apps/omnii-mobile/src/features/`:
Create `ai/api/aiApi.ts`:
- `getMeetingBriefing(eventId)` - Fetch briefing
- `getUpcomingBriefings()` - List briefings
- `getRelationshipSuggestions()` - Fetch suggestions
- `approveSuggestion(id)` - Approve relationship
- `rejectSuggestion(id)` - Reject relationship
- `getInsights()` - Fetch analytics insights
- `dismissInsight(id)` - Dismiss insight
- `getAnalyticsSummary()` - Get summary
